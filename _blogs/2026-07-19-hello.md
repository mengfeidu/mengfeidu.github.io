This is a **sample blog post**. Copy this file under `_blogs/` and edit the front matter plus body to publish a new post.

### Front matter fields

| Field | Required | Meaning |
| --- | --- | --- |
| `layout` | yes | keep as `blog` |
| `title` | yes | post title shown in the list and detail page |
| `date` | yes | sorting / display date |
| `link` | no | if set, the list page opens this external URL instead |

### Writing tips

- Use normal Markdown: headings, lists, links, and images.
- Inline images can be placed under `assets/images/` and referenced like:

```markdown
![description](/assets/images/photography/sample-cat.jpg)
```

Feel free to delete this sample once you start writing real posts.
